# stupid-calculator
Just A Stupid Calculator By WinForms/C++

## Simple Calculator (WinForms/C++)

This repository now includes a simple WinForms calculator implemented in C++/CLI.

### Features
- Basic operations: `+`, `-`, `*`, `/`
- Decimal input
- Clear button (`C`)
- Divide-by-zero handling

### File
- `Calculator.cpp` - WinForms UI and calculator logic.

### Build (Visual Studio)
1. Create a **CLR Empty Project** in Visual Studio.
2. Add `Calculator.cpp` to the project.
3. Ensure the project uses **/clr**.
4. Build and run.
